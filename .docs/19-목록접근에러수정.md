# 19-목록접근에러수정

## 목표
- 게시판(`/posts`)과 사원목록(`/employees`) 접근 시 발생하던 오류 원인을 정리하고 수정한다.

## 원인
- 목록 화면과 공통 헤더에서 문자열 인코딩 깨짐이 누적되면서 일부 마크업/문구가 손상됨
- 특히 목록 템플릿/헤더/페이지 타이틀 문자열이 혼재되어 접근 시 오류 화면으로 이어질 수 있는 상태였음

## 조치
1. 템플릿 재정리
- `layout/header.mustache`
- `post/list.mustache`
- `employee/list.mustache`

2. 컨트롤러 타이틀/문구 정리
- `post/PostController`
- `employee/EmployeeController`
- `auth/AuthController`

3. 예외 메시지 가독성 복구
- `LoginCheckInterceptor`
- `GlobalExceptionHandler`

## 결과
- 게시판/사원목록 관련 문자열과 마크업을 정상 한글 상태로 통일
- 테스트 기준 빌드/실행 검증 통과
