# 07 한글 더미데이터 및 상태 라벨

## 변경 목적
- 화면 표시와 초기 데이터의 자연스러운 한국어 사용자 경험 제공

## 적용 내용
1. 상태값 한글 라벨
- `EmployeeStatus`에 라벨 필드 추가
  - `EMPLOYED -> 재직`
  - `LEAVE -> 휴직`
  - `RESIGNED -> 퇴사`
- 템플릿에서 `{{label}}`, `{{status.label}}`로 출력

2. 부서명 한글화
- 초기 부서 데이터
  - 인사, 개발, 재무, 영업, 운영, 마케팅, 고객지원

3. 사원명 한글화
- 더미 사원명 전체를 한국식 이름으로 교체

4. 화면 라벨 정리
- 사원 목록/등록/수정/상세에서 상태 표시를 한글화
