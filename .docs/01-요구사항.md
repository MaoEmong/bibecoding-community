# 01 요구사항

## 목표
- Spring Boot 기반 사원관리 MVP를 구현한다.
- SSR(Mustache) + 세션 인증 방식으로 동작한다.

## MVP 범위
1. 인증
- 로그인/로그아웃
- 세션 기반 인증 유지

2. 사원 관리
- 사원 등록/목록/상세/수정/삭제
- 재직 상태(EMPLOYED/LEAVE/RESIGNED) 관리

3. 검색/목록
- 이름, 사번, 부서, 상태 검색
- 페이지네이션

4. 검증/예외
- DTO 애노테이션 검증
- 전역 예외 처리(@ControllerAdvice)
- 400/401/403/404/500 정책 준수

5. 테스트
- 정상 흐름
- 검증 실패
- 인증 실패
- 예외 핸들러 동작

## 제약
- Feature-based package 구조 유지
- Repository는 EntityManager 기반 구현
- Controller -> Service -> Repository 의존
- 작업일지(.context)와 개발문서(.project) 넘버링 준수
