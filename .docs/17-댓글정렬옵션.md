# 17-댓글정렬옵션

## 목표
- 댓글 목록 정렬을 선택할 수 있게 한다.

## 정렬 옵션
- `createdAsc`: 등록순
- `updatedDesc`: 최신 수정순

## 변경 내용
1. `CommentService`
- `findByPostId(postId, sessionUser, sort)` 오버로드 추가
- 정렬 comparator 분기 처리

2. `PostController`
- 상세 조회에 `commentSort` 쿼리 파라미터 추가
- 뷰 모델에 현재 정렬 상태 값 전달

3. `CommentController`
- 저장/수정/삭제 후 redirect 시 `commentSort` 유지
- 수정폼 진입 시 `commentSort` 전달

4. View
- `post/detail.mustache` 정렬 선택 링크 추가
- 댓글 저장/삭제 폼에 `commentSort` hidden 필드 추가
- 댓글 수정 링크에 `commentSort` 쿼리 포함
- `comment/update-form.mustache` 수정 요청에 `commentSort` hidden 필드 추가

5. Test
- `CommentServiceIntegrationTest`에 정렬 옵션 동작 시나리오 추가
