# 06 문구 한글화

## 목적
- 사용자 화면의 가독성을 위해 부서/사원/로그인 관련 문구와 예외 메시지를 한국어로 통일

## 변경 범위
1. 서비스 예외 메시지 한글화
- `employee/EmployeeService`
- `department/DepartmentService`
- `auth/AuthService`

2. 전역 예외 핸들러 한글화
- 페이지 타이틀: 알림/입력 오류/서버 오류
- 메시지: 입력값 확인/서버 오류 안내

3. 컨트롤러 페이지 타이틀 한글화
- `auth/AuthController`
- `employee/EmployeeController`
- `department/DepartmentController`

4. 템플릿 문구 한글화
- `layout/header.mustache`
- `layout/footer.mustache`
- `auth/login-form.mustache`
- `employee/list.mustache`
- `employee/save-form.mustache`
- `employee/update-form.mustache`
- `employee/detail.mustache`
- `department/list.mustache`

## 비고
- 코드 식별자(클래스명/메서드명/로그 키)는 기존 영어 유지
- 파일 인코딩은 UTF-8 without BOM 유지
