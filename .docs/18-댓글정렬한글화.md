# 18-댓글정렬한글화

## 목표
- 댓글 정렬/수정 관련 화면 문구를 한글로 통일한다.

## 변경 내용
1. 게시글 상세(`post/detail.mustache`)
- 댓글 정렬 라벨 한글화
  - 등록순
  - 최신수정순
- 댓글 작성/수정/삭제/목록 관련 문구 한글화
- 댓글 작성 placeholder 한글화

2. 댓글 수정 폼(`comment/update-form.mustache`)
- 제목/라벨/버튼 문구 한글화

3. 컨트롤러 페이지 타이틀
- `PostController`의 게시글 화면 타이틀 한글화
- `CommentController`의 댓글 수정 타이틀 한글화

## 비고
- 정렬 키(`createdAsc`, `updatedDesc`)는 내부 처리용으로 유지
