# 11 썸머노트 적용

## 목적
- 게시글 작성/수정에서 리치 텍스트 편집을 지원한다.

## 적용 범위
1. 게시글 작성 화면
- `src/main/resources/templates/post/save-form.mustache`

2. 게시글 수정 화면
- `src/main/resources/templates/post/update-form.mustache`

3. 게시글 상세 화면
- `src/main/resources/templates/post/detail.mustache`

## 구현 내용
1. Summernote 리소스 연결
- CSS: `/summernote/summernote-lite.min.css`
- JS: `/summernote/summernote-lite.min.js`
- 언어: `/summernote/lang/summernote-ko-KR.min.js`

2. 에디터 초기화
- 대상: `#contentEditor`
- 옵션: `height=320`, `lang=ko-KR`

3. 상세 렌더링
- 저장된 HTML을 그대로 렌더링하도록 `{{{post.content}}}` 적용

## 참고
- jQuery는 CDN(`code.jquery.com`)으로 로드한다.
