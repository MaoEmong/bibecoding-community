# 21-비로그인안내복구및로그인리다이렉트

## 목표
- 비로그인 상태 접근 시 안내문구 깨짐을 복구한다.
- 안내 확인 후 로그인 화면(`/login-form`)으로 이동되도록 수정한다.

## 원인
- 인증 예외 메시지와 전역 예외 페이지 타이틀 문자열이 손상된 상태였음.
- `UnauthenticatedException` 처리에서도 `redirectUrl`이 비어 있어 `history.back()`만 반복됨.

## 조치
1. `LoginCheckInterceptor`
- 비로그인 예외 메시지를 정상 문자열로 교체

2. `GlobalExceptionHandler`
- 안내/검증/서버오류 문구 정상화
- 인증 예외(`UnauthenticatedException`)는 `redirectUrl=/login-form`으로 분기 처리

## 결과
- 비로그인 접근 시 안내문구가 정상 출력
- 확인 후 로그인 화면으로 자동 이동
