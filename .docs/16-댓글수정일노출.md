# 16-댓글수정일노출

## 목표
- 댓글에 수정 시각(`updatedAt`)을 추가하고 화면에 표시한다.

## 변경 내용
1. 엔티티
- `comment/Comment`에 `updatedAt` 컬럼 추가 (`nullable = false`)

2. 서비스
- 댓글 등록 시 `createdAt`, `updatedAt`를 동일 시각으로 저장
- 댓글 수정 시 `updatedAt`를 현재 시각으로 갱신

3. 응답/화면
- `CommentResponse.ItemDTO`에 `updatedAt` 추가
- `post/detail.mustache`에서 댓글 생성/수정 시각 모두 노출

4. 테스트
- `CommentServiceIntegrationTest`에서
  - 등록 직후 `createdAt == updatedAt` 검증
  - 수정 이후 `updatedAt` 갱신 검증

## 비고
- 현재 댓글 `updatedAt`은 수정 시각 추적 목적이며, 정렬 정책은 기존과 동일하다.
