# 20-메뉴글자깨짐복구

## 목표
- 상단 메뉴 글자 깨짐 현상을 즉시 복구한다.

## 원인
- 공통 템플릿(`header/footer/home`)의 고정 한글 문자열이 손상된 상태로 저장되어 브라우저에서 깨진 글자로 출력됨.

## 조치
- `layout/header.mustache`, `layout/footer.mustache`, `home/main.mustache`를 안전한 HTML 엔티티 기반 텍스트로 교체
- UTF-8(no BOM)으로 정규화

## 결과
- 메뉴/헤더/푸터/메인 문구가 인코딩 환경에 영향받지 않고 정상 출력되도록 복구
