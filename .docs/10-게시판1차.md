# 10 게시판1차

## 목표
- 사내 커뮤니티 핵심 기능으로 게시판 CRUD 도입

## 제공 기능
1. 게시글 작성/목록/상세/수정/삭제
2. 제목/내용/작성자 키워드 검색
3. 작성자 권한 정책
- 작성자 본인 또는 관리자만 수정/삭제 가능

## 라우팅
- `GET /posts`
- `GET /posts/save-form`
- `POST /posts/save`
- `GET /posts/{id}`
- `GET /posts/{id}/update-form`
- `POST /posts/{id}/update`
- `POST /posts/{id}/delete`

## 데이터
- `posts` 테이블 추가
- 작성자 FK: `posts.user_id -> users.id`

## 비고
- 목록/상세 출력은 작성자 이름/이메일 포함
- 추후 댓글, 공지 분리, 파일첨부로 확장 가능
