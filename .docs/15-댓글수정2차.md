# 15-댓글수정2차

## 목표
- 댓글 수정 기능을 추가해 작성자/관리자가 댓글 내용을 변경할 수 있게 한다.

## 변경 내용
1. 요청/응답 모델
- `CommentRequest.UpdateDTO` 추가
- `CommentResponse.EditFormDTO` 추가

2. 서비스
- `findForUpdate(postId, commentId, sessionUser)` 추가
- `update(postId, commentId, request, sessionUser)` 추가
- 정책: 댓글 작성자 또는 관리자만 수정 가능
- 입력 정제: `Jsoup.clean(..., Safelist.none())`

3. 컨트롤러
- `GET /posts/{postId}/comments/{commentId}/update-form`
- `POST /posts/{postId}/comments/{commentId}/update`

4. 화면
- `templates/comment/update-form.mustache` 추가
- `templates/post/detail.mustache` 댓글 수정 링크 추가

5. 테스트
- `CommentServiceIntegrationTest`에 수정 권한/수정 결과 시나리오 추가

## 후속
- 댓글 수정 이력(수정일, 수정자) 관리 필요 시 3차에서 확장
