# 02 설계

## 패키지 구조
- com.example.bibecoding01.auth
- com.example.bibecoding01.employee
- com.example.bibecoding01._core

## 인증 설계
- 세션 키: `LOGIN_USER`
- 로그인 경로: `/login-form`, `/login`, `/logout`
- 인터셉터에서 인증 검사
- 예외 URL: `/login-form`, `/login`, `/h2-console/**`, 정적 리소스

## 도메인 설계
1. Employee
- id
- employeeNo(고유)
- name
- email
- department
- position
- status
- hiredAt

2. EmployeeStatus(enum)
- EMPLOYED
- LEAVE
- RESIGNED

## 레이어 설계
1. Controller
- 요청 바인딩, 검증, 뷰 반환

2. Service
- 트랜잭션, 도메인 검증(사번 중복 등)

3. Repository(EntityManager)
- Optional 반환 규칙
- 검색/페이지 조회 메서드 제공

## 화면 설계
- 로그인 페이지
- 사원 목록 페이지(검색/페이지 링크)
- 사원 등록/수정 폼
- 사원 상세 페이지
- 공통 에러 알림 페이지

## 예외 설계
- BadRequestException(400)
- UnauthenticatedException(401)
- ForbiddenException(403)
- NotFoundException(404)
- GlobalExceptionHandler에서 일괄 처리
