# 13-댓글1차

## 목표
- 게시글 상세 화면에서 댓글 등록/조회/삭제를 지원한다.
- 댓글 삭제 권한은 작성자 본인 또는 관리자만 허용한다.

## 적용 범위
1. 도메인 추가
- `comment/Comment`
- `comment/CommentRepository`, `comment/CommentRepositoryImpl`
- `comment/CommentRequest`, `comment/CommentResponse`
- `comment/CommentService`, `comment/CommentController`

2. 게시글 상세 연동
- `post/PostController` 상세 조회에 댓글 목록/등록 폼 모델 추가
- `templates/post/detail.mustache` 댓글 입력/목록/삭제 버튼 추가

3. 보안/권한
- 댓글 본문은 `Jsoup.clean(..., Safelist.none())`로 정제
- 삭제 시 작성자 본인 또는 관리자만 허용

4. 테스트
- `comment/CommentServiceIntegrationTest`
- 작성자/관리자 삭제 허용, 타인 삭제 차단, 댓글 XSS 정제 검증

## 후속 고려사항
- 게시글 삭제 시 댓글도 함께 삭제되도록 연동 정책 보완
- 댓글 수정 기능 2차 추가
